========
Commands
========

Draft commands can be split into three buckets:

1. **Create** a project
2. **Format** a project's file content
3. **Maintain** a project

Create
------

- ``create-project``: content
- ``parse``: content

Format
------

- ``split``: content
- ``trim``: content

Maintain
--------

- ``sequence``: content
- ``stats``: content
- ``compile``: content
- ``outline``: content
